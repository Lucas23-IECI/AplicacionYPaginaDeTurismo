# Contexto Base - PuroCode

## Que es PuroCode

PuroCode es una **Agencia de Desarrollo Web & SaaS Premium en Chile**. Crea landing pages, sitios corporativos, e-commerce y soluciones SaaS a medida. Utiliza tecnologias como Next.js y React.

### Paginas publicas

- Home
- Formulario de cotizacion (briefing para elegir entre landing page, sitio corporativo o e-commerce, con cotizacion instantanea y gratuita)
- Politica de privacidad
- Terminos y condiciones

### Posicionamiento

- Agencia web chilena de posicionamiento premium
- Canal comercial y de captura de leads
- Tres ofertas principales para cliente no tecnico: landing page, web corporativa, e-commerce
- Linea avanzada de SaaS a medida

### Stack tecnico actual de PuroCode (plataforma principal)

- Next.js + TypeScript + Tailwind + Prisma
- Turso en produccion, SQLite en local
- i18n custom, pricing engine, generacion de DOCX/XLSX
- Auth admin con JWT, portal de clientes con magic links
- Sistema de correos con varios fallbacks
- Promociones, descuentos, multiidioma, multimoneda
