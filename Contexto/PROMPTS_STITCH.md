# Prompts para Stitch (Google Design with AI)

## Contexto

Estos prompts son para generar el diseno visual del HOME de la web corporativa de la plataforma de turismo. NO es un pitch deck, NO es una app, NO es un dashboard. Es la preview visual de como se vera la solucion.

---

## Prompt largo (recomendado)

```
Design a premium responsive homepage for a tourism and local activities web platform. This is a desktop and mobile responsive website interface preview, not a presentation, not a pitch deck, not a SaaS dashboard, and not a mobile app.

The design must show how the main public homepage of the platform would look visually. Focus on the product interface itself, not on marketing copy or business explanations.

The homepage should feel modern, polished, elegant, premium, and highly visual. It should combine a corporate-quality digital product aesthetic with tourism discovery elements. Clean layout, refined spacing, soft cards, map-based UI hints, polished filters, and strong visual hierarchy.

Main idea of the interface:
A user enters the site and can immediately explore nearby activities, events, fairs, festivals, concerts, gastronomy, and cultural experiences. The homepage should visually suggest that the platform is curated, organized, and high quality.

Include these visual sections in the homepage:

A clean top navigation bar with logo, menu, login access, and a highlighted button for advertisers

A hero section with a strong visual search experience

A search module with:

location field

distance in kilometers selector

date selector

category selector

A premium "featured activities" section with only a few highlighted event cards, not too many

Event cards should look elegant and curated, showing:

image

title

city

date

category

distance

A visual calendar preview section for browsing activities by date

A browse by city or region section

A large featured event section with image, summary, map preview, and event details

A visual block that suggests organizers can submit activities

A visual block that suggests admin review / approval exists before publication

A footer with clean corporate styling

Important design constraints:

Do not make it look like a generic landing page full of sales copy

Do not make it look like a marketplace with dozens of cards

Do not turn it into an app UI

Do not create dashboard screens

Do not overuse text blocks

Prioritize interface preview, layout, visual hierarchy, and product feel

Use realistic UI sections and polished content blocks

Keep the homepage elegant, uncluttered, and premium

Style direction:

premium digital product

tourism + discovery + curated experiences

modern corporate look

editorial-quality visuals

subtle map and geolocation inspiration

soft shadows, rounded cards, refined spacing

strong typography

visually impressive but clean

Color palette:

sophisticated and modern

white or light neutral base with accents in deep green, teal, blue, or warm sand

professional and premium, not playful

avoid excessive neon or childish colors

Typography:

modern sans-serif

bold and elegant headings

highly readable interface text

Use realistic homepage content in Spanish for labels, cards, filters, navigation, and sections.
```

---

## Prompt corto (alternativo)

```
Design a premium responsive homepage for a tourism and local activities web platform. Show the actual public-facing product homepage, not a presentation, not a pitch deck, not a dashboard, and not a mobile app. Include a top navbar, a hero with a search module for location, kilometers, date, and category, a few elegant featured activity cards, a calendar preview, browsing by city/region, a large featured event section with map preview, a visual section suggesting organizers can submit activities, and a subtle section suggesting admin curation before publication. The page should feel modern, clean, premium, curated, and visually polished, with a corporate-quality digital product aesthetic. Use realistic Spanish UI text. Avoid too much marketing copy, avoid too many event cards, and avoid clutter.
```
