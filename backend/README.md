# Backend — Plataforma de Turismo

## Stack definido

- **Supabase**: Auth, base de datos PostgreSQL, almacenamiento multimedia
- **Pagos**: Stripe o Transbank (por definir)
- **IA editorial**: Claude API (asistente de redaccion para curaduria)

## Modulos planificados

### Portal de anunciantes
- Auth / login
- Gestion de suscripcion
- Carga de eventos (textos, imagenes, fecha, hora, lugar)
- Estado de eventos (pendiente / aprobado / rechazado)
- Historial de publicaciones
- Metricas basicas (futuro)

### Panel de administracion
- Dashboard de solicitudes entrantes
- Flujo de aprobacion / rechazo
- Edicion manual de contenido
- Asistente IA para mejorar redaccion
- Gestion de anunciantes
- Moderacion de imagenes
- Estadisticas
